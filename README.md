**Google Search UI Clone**

This is a simple Google-style search page built with HTML, CSS and JavaScript. It works like a lightweight front end that redirects your search queries to Google.

**Features**
1.Clean interface inspired by Google
2.Search bar with hover and focus effects
3.Redirects searches to Google results
4.Mobile friendly layout

**Files in this project**

1.index.html – main layout
2.style.css – page styling
3.script.js – search handling logic

**How to use**
1.Open the page in any browser.
2.Type your search term.
3.Click Search or press Enter.
