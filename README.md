Google Search UI Clone

This is a simple Google-style search page built with HTML, CSS and JavaScript. It works like a lightweight front end that redirects your search queries to Google.

Features

Clean interface inspired by Google

Search bar with hover and focus effects

Redirects searches to Google results

Mobile friendly layout

Files in this project

index.html – main layout

style.css – page styling

script.js – search handling logic

How to use

Open the page in any browser.

Type your search term.

Click Search or press Enter.
